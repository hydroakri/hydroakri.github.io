我的电脑双硬盘双系统，使用Windows 11 + ArchLinux
在Windows上使用 scoop + winget / Archlinux 下使用 paru+pacman
我尽可能使用开源/免费

全平台#

  • 网络加速：Watt Toolkit
    提供了多网站的网络加速 不局限于steam 我一般用来加速github 和 steam
  • 编辑器：Neovim
    编辑器之神 没什么好说的
  • 本地视频播放： mpv
    轻巧的一款本地播放器 支持字幕 我拿它来放电影看
  • 视频录制：OBS
  • 手机投屏：scrcpy
    基于命令行 连上手机开启adb调试以后敲命令就行 没有多余的依赖
  • 流媒体：YesPlayMusic（网易云）,Cider(Apple Music),Spotify
    用 YesPlayMusic 多一点，因为它有原版网易云不支持的“连接到Last.FM”功能
  • GIT：LazyGit
    我的水平还不足以完全开发这个软件 不常用
  • 配置文件管理：GNU STOW
    非常好用的配置文件管理，用软连接的方式链接到文件所在位置，源文件集中在一起，非常方便
  • 系统监视器：Btop/Bottom
  • 截图：Flameshot
    上游还没修Wayland下面的bug TAT
  • 种子下载：Motrix
    非常好用 自带tracker list 即使它是electron我用能接受
  • 代码分享：Silicon
  • 终端：wezterm
  • 密码管理：KeepassXC
  • 隔空投送：Localsend
    好用，尤其是在AP隔离的校园网环境下

Windows#

  • 把Windows 11 改为10的模样：ExplorerPatcher
  • 使游戏窗口全屏显示：Magpie
  • 管理“此电脑”里删不掉的流氓“快捷方式”：MyComputerManager
  • 平铺管理器：komorebi
  • 系统优化：optimizer, Dism++
  • 让Windows拥有macOS的触发角功能: hotcorner
  • Windows免费激活：Microsoft-Activation-Scripts
  • 系统信息：systeminformer
  • 桌面文件夹整理：TaskbarX
  • 内存优化：memreduct
  • WSL管理：LxRunOffline
  • BTRFS: btrfs
